# Hướng dẫn đưa Portfolio lên Google Sites (phong cách Coquette)

Bạn đã có sẵn 3 trang HTML trong thư mục `f:\portfolio`:

| File | Nội dung |
|------|----------|
| `index.html` | Giới thiệu |
| `bai-tap.html` | Bài tập (tuần 1–7) |
| `tong-ket.html` | Tổng kết trải nghiệm |

## Cách 1 — Nhanh: Mở file HTML trên trình duyệt, chụp / copy nội dung

1. Double-click `index.html` → mở bằng Chrome/Edge.
2. Trên [Google Sites](https://sites.google.com), tạo site mới.
3. Tạo **3 trang** trong menu: *Giới thiệu*, *Bài tập*, *Tổng kết*.
4. Copy từng đoạn văn từ HTML sang khối văn bản trên Google Sites.
5. Chọn theme màu **hồng / kem** (tùy chỉnh màu nền #FFF5F8, chữ #5C4A52).

## Cách 2 — Nhúng toàn bộ giao diện (gần giống bản HTML)

Google Sites cho phép **Embed → Embed code** (chỉ trên site nội bộ hoặc khi bật embed).

1. Đăng tải thư mục `portfolio` lên **GitHub Pages**, **Netlify** hoặc **Google Drive** (public link).
2. Trên mỗi trang Google Sites, thêm khối **Embed** → dán URL:
   - `https://your-site.netlify.app/index.html`
   - `https://your-site.netlify.app/bai-tap.html`
   - `https://your-site.netlify.app/tong-ket.html`
3. Menu Google Sites trỏ tới 3 URL tương ứng.

## Cách 3 — Chỉ dùng Google Sites (không host HTML)

1. **Trang Giới thiệu:** copy nội dung từ `index.html` (phần hero + 3 card).
2. **Trang Bài tập:** copy từ `bai-tap.html`; với Tuần 1–5 và infographic, **Insert → Image** từ `LMS.pdf`.
3. **Trang Tổng kết:** copy từ `tong-ket.html`.
4. **Navigation:** Site settings → Pages → bật navigation, sắp 3 trang.
5. **Giao diện coquette trên Sites:**
   - Font tiêu đề: *Playfair Display* hoặc *Cormorant*
   - Font nội dung: *Quicksand* / *Nunito*
   - Màu chủ đạo: hồng pastel #F8D7E3, nền kem #FFFAF7

## Ảnh từ LMS.pdf

Các tuần 1–5 và infographic tuần 6, flowchart tuần 7 trong PDF — export ảnh rồi thay các ô “Chèn ảnh…” trên trang Bài tập.

## Xem trước local

Mở `index.html` trong trình duyệt; menu trên cùng chuyển giữa 3 trang.
